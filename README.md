Diabetes Patients Analysis Dashboard 🏥📊
Project Overview
This Power BI dashboard provides a comprehensive analysis of hospital secondary care data for diabetes patients. The project aims to help healthcare providers and hospital management understand patient demographics, clinical trends, and factors influencing hospital readmission.

 Key Features & Insights
The dashboard is organized into three specialized views:

1. Executive Overview (Hospital Snapshot):
   - High-level KPIs: Total Encounters, Readmission Rate, and Unique Patient Count.
   - Analysis of Readmission Risk across different Medical Specialties.
   - Demographic breakdown (Age & Race) vs. Readmission rates.

2. Clinical Insights:
   - Deep dive into clinical activities: Average Lab Procedures and Medication usage.
   - Correlation between Length of Stay (Days in Hospital) and Readmission.
   - Patient distribution by Medical Specialty.

3. Patient Records & Drill-down:
   - Interactive Patient Lookup: Search for specific patients by their ID.
   - Dynamic Patient Profile: View detailed clinical history (Age, Gender, Specialty, Lab Tests) for selected individuals.
   - Visual comparison of Clinical Activity Volume using Funnel and Gauge charts.

Technical Skills Applied
- Data Modeling:Established relationships between complex healthcare datasets.
- DAX Measures: Created custom calculations for Readmission Rates, Averages, and Dynamic Titles.
- Advanced Visualization:Implemented Conditional Formatting, Multi-row Cards, and Gauge Charts.
- Interactive UI/UX: Built a seamless navigation system with custom buttons and slicers.
- Smart Narratives: Integrated AI-driven summaries to provide instant clinical context.

How to Use
1. Download the `.pbix` file.
2. Open it using Power BI Desktop.
3. Use the navigation buttons on the right to switch between different views.
4. In the 'Patient Records' tab, use the search slicer to find specific patient data.
